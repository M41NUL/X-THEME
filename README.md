## What is X-THEME?

**X-THEME** is a full Termux customization tool written in pure Shell Script. Change your terminal color theme, font, startup banner, prompt style, and extra keyboard layout ‚ÄĒ all from one interactive menu. Supports both bash and zsh. Includes backup and restore.

> Make your Termux look exactly the way you want.

## Installation

### One-line install

```bash
git clone https://github.com/M41NUL/X-THEME.git && cd X-THEME && bash installer.sh
```

### Step by step

```bash
# Step 1 ‚ÄĒ Clone
git clone https://github.com/M41NUL/X-THEME.git
cd X-THEME

# Step 2 ‚ÄĒ Install & launch
bash installer.sh
```

The installer will:
- Update Termux packages
- Install `curl`, `git`, `termux-tools`
- Request Android storage permission (once only)
- Auto-launch X-THEME after install

### Run manually (after first install)

```bash
cd X-THEME
bash x-theme.sh
```
